# pve-tools
Proxmox VE Tools (CLI helper untuk CT/VM)

## Cara Penggunaan:
Buat file script:
```bash
git clone https://github.com/abdurrozakskom/pve-tools.git
cd pve-tools
```

## Beri permission executable:
```bash
chmod +x pve-tools.sh install-pve-tools.sh
```

## Install tools:
```bash
sudo ./install-pve-tools.sh
```

## Jalankan tools:
```bash
pve-tools
```
